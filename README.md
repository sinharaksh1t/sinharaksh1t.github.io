# Personal Website

Small personal website built with Next.js, React, TypeScript, and Tailwind CSS.

## Routes

- `/`
- `/contact`
- `/projects`
- `/projects/project-1`
- `/projects/project-2`
- `/projects/project-3`

## Getting Started

Run:

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
```

This project is configured for static export so GitHub Pages can deploy the generated `out/` directory.

## Lint

```bash
npm run lint
```

## Notes

- `PRD.md` contains the product brief.
- Contact/project content is placeholder-first and easy to swap later.
